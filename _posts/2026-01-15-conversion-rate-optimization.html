Step 1: diagnose with evidence

  • Funnel drop-off analysis
  • Landing page engagement metrics
  • Heatmaps and recordings
  • Form analytics

Step 2: prioritize with impact and effort

I rank tests by expected impact, effort, and confidence based on data quality.

Step 3: write a clear hypothesis

"If we change X, then Y will improve because Z." If I cannot write that, the test is not ready.

Step 4: run clean experiments

  • Single major variable per test
  • Statistical sample sizes
  • Run long enough for weekly patterns

Step 5: document and loop

Wins and losses both feed the AI reporting layer. Documentation compounds over time.
